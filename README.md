# ☕ MOMO COFFEE

모모커피(MOMO COFFEE)는 사용자가 다양한 커피와 디저트를 주문할 수 있는 카페 주문 웹 애플리케이션입니다.

고객은 메뉴를 조회하고 장바구니에 담아 주문할 수 있으며, 관리자는 메뉴와 주문을 관리할 수 있습니다.

실제 카페 주문 서비스를 가정하여 사용자와 관리자의 기능을 분리하고 CRUD 기반의 화면 설계를 연습하는 것을 목표로 개발했습니다.

---

# 1. 프로젝트 목표

- 카페 주문 서비스를 웹으로 구현
- 사용자와 관리자 기능을 역할별로 분리
- 메뉴 조회부터 주문까지의 흐름 구현
- 관리자 메뉴 및 주문 관리 기능 구현
- 실제 서비스와 유사한 폴더 구조 및 화면 설계 학습

---

# 2. 주요 사용자

| 사용자 | 주요 목적 |
|---------|-----------|
| 고객(Customer) | 메뉴 조회, 장바구니 관리, 주문 |
| 관리자(Admin) | 메뉴 관리, 주문 관리 |

---

# 3. 핵심 기능

## 고객

### 메뉴

- 메뉴 목록 조회
- 메뉴 상세 조회
- 카테고리별 메뉴 조회

### 장바구니

- 메뉴 담기
- 수량 변경
- 메뉴 삭제
- 전체 비우기

### 주문

- 주문 생성
- 주문 완료 화면

### 마이페이지

- 내 정보 조회
- 최근 주문 내역 확인
- 주문 현황 확인

---

## 관리자

### 메뉴 관리

- 메뉴 목록 조회
- 메뉴 등록
- 메뉴 수정
- 메뉴 삭제

### 주문 관리

- 주문 목록 조회
- 주문 상세 조회
- 주문 상태 확인

### 대시보드

- 주문 현황
- 최근 주문
- 매출 요약(예시 데이터)

# 8. 실행 방법


```bash
cd new-cafe-app
```

VS Code의 Live Server 또는 원하는 웹 서버를 이용해 실행합니다.

또는 `index.html`을 브라우저에서 직접 실행할 수 있습니다.
